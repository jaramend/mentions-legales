Generate a shortcode with the legal mentions, which are mandatory for French pro websites.

== Description ==

This plugins allow the administrator of a Wordpress site to display the French legal mentions with the use of a shortcode.
The user has just to write site informations and to put the shortcode on a new "Mentions légales" Wordpress page.
The plugin is only in French because other countries don't need legal mentions.

= Usage =
 1. Go to Options -> Mentions légales
 2. Fill in all the form with the appropriate informations
 3. Copy the shortcode which appears after the save, and paste it on a new page "Mentions légales"

= Translations available =

 * French

== Installation ==

Use directly the Wordpress dashboard :
 1. Go to the *Plugins* menu and click *Add New*.
 2. Search for *Mentions légales*.
 3. Click *Install Now* next to the *Mentions légales* plugin.
 4. Activate the plugin.

== Frequently Asked Questions ==

= Why this plugin is only in French ? =

This plugin is only in French because it generates legal mentions only for French  websites.
